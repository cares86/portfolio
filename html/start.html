Intro

Als ambitieus persoon ben ik altijd opzoek naar kennis en skills die ik kan gebruiken voor mijn eigen intresses of voor een baan. Door mijn zeer brede intresses weet ik van veel een basiskennis en blijf ik steeds meer nieuwe uitdagingen opzoeken. Op dit moment is dat de basiskennis opdoen van HTML, CSS en JAVASCRIPT. de dingen dit hier mee te maken hebben zoals verschillende vormen van CSS en libraries focus ik me ook op. Eigenlijk alles op gebied van front-end en design. In mijn vrije tijd ben ik veel bezig met creatieve uitingen. Zo houd ik me bezig met tekenen, schilderen en fotografie. Ook fotoshopping en video editting houd me bezig verder ben ik een natuurmens wat ok veel terug te vinden is in mijn werk. Verder ben ik ook een ware gamer ben ik gek op gadgets en technologische hoogstandjes intresses ik me ook zeker in de huidige ontwikkeling hiervan.

Wat ik kan en weet

Ik ben vanaf mijn 15e altijd actief bezig geweest op de arbeidsmarkt. Ik heb over de jaren heen leuke baantjes gehad en heb in veel verschillende bedrijfstakken ervaring op gedaan. Ik heb altijd met veel plezier terug gekeken naar mijn vorige werkplekken. Na een pauze op de arbeidsmarkt zocht ik naar iets wat dicht bij mij ligt. Ik kwam gaandeweg terecht bij Code Gorrilla. Bij hun kreeg ik de kans om iets nieuws te proberen want dicht bij mijn eigen intresses en talenten ligt. Vanaf het moment dat ik ben gaan leren bij Code Gorrilla heb ik een aantal code languages aangeleerd. Ik heb drie geleerd die ik vanaf dat punt perfectioneer. Wanneer ik een redelijke basiskennis heb geleerd vul ik dit aan op mijn charts. Wanneer ik een nieuwe code language heb geleerd krijgt deze op deze pagina ook zijn eigen chart met het percentage van beheersing over deze lanquage.
